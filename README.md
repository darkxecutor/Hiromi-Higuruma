> **"Justice and code are my primary domains. I don't tolerate inefficiency, and I build interfaces with the precision of a legal argument."**

A modern, anime-inspired portfolio website featuring a **judicial UI theme** with a cinematic intro screen, smooth animations, and a professional layout. Designed to showcase skills, projects, and services with a unique aesthetic inspired by Hiromi Higuruma from Jujutsu Kaisen.


---

## ✨ Features

### 🎬 Cinematic Intro Experience
- **System-ready splash screen** with animated orbital elements
- Smooth fade-out transition to main content
- Live status indicators and system metrics

### 🎨 Modern UI Design
- **Dark theme** with deep green-gray color palette
- Glassmorphism effects with backdrop blur
- Responsive layout that works on all devices
- Custom animations (slide, fade, orbit rotation)

### 📱 Complete Sections
- **Home** – Hero section with personal introduction and call-to-action
- **About** – Skills, expertise, and personal domain focus
- **Projects** – Grid layout showcasing 6+ portfolio projects
- **Services** – Two core service cards with feature tags
- **Contact** – Functional contact form with social links

### ⚡ Interactive Features
- Smooth scroll navigation with active link highlighting
- Scroll-triggered reveal animations
- Interactive buttons with hover effects
- Form submission with confirmation dialog

📱 Responsive Design

The portfolio is fully responsive and adapts to:

    Desktop (1200px+): Full layout with side-by-side sections

    Tablet (768px-1199px): Adjusted padding and column layouts

    Mobile (<768px): Stacked sections, reduced font sizes, condensed navigation
